## Description

Detective Lineup is a game that reflects the hallucinations and social cue aspects of Schizophrenia. The game simulates an experience in which the player has to determine the correct suspect across 3 levels, each progressively harder with added suspects and game mechanics.

The player navigates through the game by learning motives and additional information about each suspect. They must convict the correct suspect in order to move to the next level using game mechanics such as visual hallucinations, audio and the notebook for additional field notes.

Using all these features the player navigates the 3 different cases to determine the suspect through visual clues, audio clues and field notes all before the time runs out.

## Setup and Interaction Instructions

How to Run the Game:

1. Download or clone the project folder.
2. Open the project in VS Code or any code editor.
3. Run the project using Live Server or open the index.html file in a browser.
4. The game will load in the browser window.
5. The game uses the p5.js library for rendering graphics and handling interaction.

How to play the game:

Level 1:

1. Click the Start Game button to begin
2. Read over the case file and click Begin
3. Read through the controls guide to learn about the game mechanics and click Start Level
4. Click on a person to begin inspecting
5. Click the Magnify button to reveal the suspects real emotions
6. Click through each player until you reach a verdict
7. Click the convict button and then click who you think is the suspect to pass the level

Level 2:

1. Same step 1-4 from Level 1.
2. Click the ask button to get an audio statement from each suspect
3. Click the board button on the main lineup page to compare evidence.
4. Same step 6-7 from Level 1

Level 3:

1. Same step 1-4 in level 1 and 2-3 in Level 2
2. Click the Notebook button to get field notes from each suspect
3. Same step 6-7 from Level 1

Win Conditions

1. Once you complete all the levels, you will be lead to a Win screen
2. Click anywhere on the screen to return back to the start screen

Lose Conditions

1. If you convict the wrong suspect you will be lead to a Try Again screen
2. Click the Retry button to play the level again or the Home button to return back to the start screen

## Iteration Notes

Post-Playtest:

Timer: Include a timer for each level so that if the player does not convict a suspect in time they have to try again

Visual Cues: When you click the Magnify button, the facial expression of the suspect changes to their real emotion. You are only limited to press this button a few times depending on the level.

Audio Cues: When you click the ask button, instead of a statement showing up, the player hears an audio statement from the suspect.

Instructions Button: A small question mark button on each level screen that explains the different game mechanics included in that level.

Fixed UI:Fixed the overall UI so buttons and images better align.
